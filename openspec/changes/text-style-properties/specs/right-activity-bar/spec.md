## 修改需求

### 需求:右侧内容面板默认隐藏并可通过图标切换
右侧内容面板必须默认隐藏；当用户点击图标列中的图标时展开并显示对应分页，再次点击当前激活图标时必须收起。**当属性分页处于激活状态且选中图层为 text 时，面板必须展示字体、字号、颜色三项文本样式属性。**

#### 场景: 点击未激活图标展开
- **当** 用户点击右侧图标列中未激活图标
- **那么** 右侧内容面板展开并显示该图标对应分页

#### 场景: 点击已激活图标收起
- **当** 用户点击右侧图标列中当前激活图标
- **那么** 右侧内容面板收起并仅保留图标列

#### 场景: 展开时恢复最近宽度
- **当** 右侧内容面板处于收起状态后再次被展开
- **那么** 面板必须以最近一次记录宽度展开

#### 场景: 属性分页中 text 图层展示样式属性
- **当** 右侧当前分页为属性且选中图层为 text
- **那么** 面板在基础属性之外展示字体、字号、颜色三项文本样式属性
