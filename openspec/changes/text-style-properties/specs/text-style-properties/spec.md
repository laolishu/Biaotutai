## 新增需求

### 需求:text 类型必须展示字体、字号、颜色
当选中图层类型为 text 时，右侧属性栏必须展示字体（fontFamily）、字号（fontSize）、颜色（color）三项样式属性。

#### 场景:text 图层显示样式属性
- **当** 用户选中 `type === "text"` 的图层
- **那么** 属性栏显示字体、字号、颜色三项内容

#### 场景:样式字段缺失时降级展示
- **当** text 图层任一样式字段缺失或为空
- **那么** 对应项必须显示统一占位值（如 `--`），且页面禁止报错

### 需求:非 text 类型禁止显示文本样式区块
系统必须确保文本样式属性区块仅在 text 类型图层下展示。

#### 场景:选中非 text 图层
- **当** 用户选中 `type !== "text"` 的图层
- **那么** 文本样式属性区块不显示
