## 新增需求

### 需求:右侧属性栏必须展示 image 缩略图属性
当选中图层类型为 image 时，系统必须在右侧属性栏渲染缩略图属性区块，并提供预览状态信息。

#### 场景: 选中 image 图层
- **当** 用户选中的图层 `type === "image"`
- **那么** 右侧属性栏显示缩略图属性区块，包含缩略图预览与尺寸信息

#### 场景: 选中非 image 图层
- **当** 用户选中的图层 `type !== "image"`
- **那么** 右侧属性栏禁止显示 image 缩略图属性区块

### 需求:image 缩略图必须支持点击放大预览
image 缩略图必须支持点击触发放大预览，用户可查看更大图像内容。

#### 场景: 点击缩略图预览
- **当** 用户点击右侧属性栏中的 image 缩略图
- **那么** 系统显示放大预览视图

#### 场景: 预览资源不可用
- **当** image 图层缺少可用预览资源或加载失败
- **那么** 系统显示占位图与失败提示，且属性栏其他信息仍可正常查看
