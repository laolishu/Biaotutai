## 新增需求

### 需求:属性面板展示选中图层属性
右侧属性面板必须展示当前选中图层的基础属性，当无选中图层时展示空状态提示。

#### 场景:选中图层后展示属性
- **当** store.selectedLayerId 不为 null
- **那么** 属性面板展示对应图层的 name、type、x、y、width、height 属性值

#### 场景:无选中图层时展示提示
- **当** store.selectedLayerId 为 null
- **那么** 属性面板展示"请选择图层"提示文本，不报错

### 需求:属性值支持一键复制
用户必须能点击属性值将其复制到剪贴板。

#### 场景:点击属性值复制
- **当** 用户点击某个属性的数值区域
- **那么** 该数值被写入系统剪贴板，并短暂显示"已复制"反馈

### 需求:文本图层展示附加样式属性
当选中图层 type 为 `text` 时，属性面板还必须展示文本相关样式属性。

#### 场景:文本图层展示字体信息
- **当** 选中图层 type 为 `text` 且 style 字段包含字体信息
- **那么** 属性面板额外展示 fontFamily、fontSize、color 字段
