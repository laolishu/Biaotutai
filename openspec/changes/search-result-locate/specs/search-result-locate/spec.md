## 新增需求

### 需求:搜索结果项双击定位
`LayerSearchResults` 组件中的每个搜索结果项必须响应双击（`dblclick`）事件。双击时系统必须执行以下操作：
1. 调用 `store.selectLayer(layer.id)` 选中该图层
2. 调用 `store.setSearchQuery('')` 清空搜索内容，使图层面板回到完整图层树视图
3. 清空搜索后，已有的 `LayerItem` `scrollIntoView` 逻辑必须自动将图层树滚动到选中图层位置

#### 场景:双击搜索结果选中并清空搜索
- **当** 用户在搜索结果列表中双击某图层项
- **那么** 该图层被选中，搜索框内容被清空，图层树视图显示并滚动定位到该图层

#### 场景:双击不影响单击行为
- **当** 用户在搜索结果列表中单击某图层项
- **那么** 仅选中该图层，搜索框内容保持不变，搜索结果列表继续显示

#### 场景:清空搜索后图层树可见
- **当** 双击触发后搜索内容被清空
- **那么** 图层面板切换回完整图层树（`store.searchQuery` 为空时显示树视图）
