## 修改需求

### 需求:所有页面颜色必须引用 CSS 变量
Upload.vue、Workbench.vue、Viewer.vue 中的 `<style scoped>` 块内所有颜色声明（背景色、边框色、文字色、强调色）必须替换为对应的 `var(--color-*)` 引用，禁止保留硬编码颜色值（静态演示数据中的内联 style 除外）。在亮色主题下，Header 图标按钮相关边框与悬停反馈必须使用 token 提供可辨识对比度，禁止出现边框不可见或交互态不明显的情况。

#### 场景: 主题切换时所有页面颜色更新
- **当** `data-theme` 属性在暗色与亮色之间切换
- **那么** 三个页面的所有主题相关颜色必须立即反映新主题，无需刷新页面

#### 场景: 亮色主题下发光效果降级
- **当** `data-theme` 为 `light`
- **那么** 原暗色主题的 glow `box-shadow` 效果必须以降低不透明度或去除的方式降级，避免在白色背景上显示突兀

#### 场景: 亮色主题下图标按钮边框与悬停可辨识
- **当** `data-theme` 为 `light` 且用户观察或悬停 Header 图标按钮
- **那么** 按钮边框与悬停背景必须具备清晰可见的层次差异
