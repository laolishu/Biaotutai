## 新增需求

### 需求:搜索输入框始终可见
图层面板头部必须包含一个搜索输入框，无论图层面板是否展开，只要面板打开就必须显示该输入框。

#### 场景:面板打开时搜索框可见
- **当** 图层面板处于展开状态
- **那么** 搜索输入框在 panel-head 下方显示，占满面板宽度

#### 场景:搜索框支持键盘清空
- **当** 用户在搜索框中按下 Esc 键
- **那么** 输入框内容被清空，图层面板恢复完整树视图

#### 场景:搜索框支持点击清空
- **当** 搜索框有内容且用户点击 × 清空按钮
- **那么** 输入框内容被清空，图层面板恢复完整树视图
