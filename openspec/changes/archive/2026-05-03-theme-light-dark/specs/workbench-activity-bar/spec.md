## 新增需求

### 需求:Workbench Header 必须提供主题切换入口
Workbench 页面的顶部 Header 右侧区域必须提供一个主题切换按钮，使用太阳/月亮图标标识当前主题状态，点击后切换到对方主题。

#### 场景: 暗色主题时显示切换入口
- **当** 当前主题为 `dark`
- **那么** Header 右侧必须显示表示"切换到亮色"的图标按钮（太阳图标）

#### 场景: 亮色主题时显示切换入口
- **当** 当前主题为 `light`
- **那么** Header 右侧必须显示表示"切换到暗色"的图标按钮（月亮图标）

#### 场景: 点击按钮触发主题切换
- **当** 用户点击主题切换按钮
- **那么** 必须调用 `toggleTheme()`，页面主题立即切换，按钮图标随之更新
