提案：上传文件页 - 默认勾选“导入时忽略隐藏图层”

## 概述

在上传 PSD/图像文件时，上传文件页应在导入选项中将“导入时忽略隐藏图层”（或类似文案）复选框默认设为已勾选。此变更将新增或调整上传界面和导入流程的默认值，以减少不必要地导入大量隐藏图层带来的噪声。

## 为什么要做

- PSD 文件中常含大量用于排版或备用的隐藏图层，默认导入这些图层会增加画布复杂度并影响后续处理。
- 在绝大多数场景下，用户期望只导入可见内容；将该选项默认开启能提升用户体验并降低误操作成本。

## 范围

- 前端：上传文件页 UI，新增或调整复选框的默认状态为“已勾选”。
- 导入流程：确保该默认选项将 `skipHiddenLayers`（或等价参数）传入 PSD 解析/规范化流程。
- 文档：在导出/导入规范文档中更新说明。

## 验收标准

- 打开上传文件页，新导入流程中“导入时忽略隐藏图层”复选框为已勾选状态。
- 导入 PSD 时，当该选项为勾选，隐藏图层不会出现在最终图层列表与导出包中；未勾选则行为与当前实现一致。
- 相关单元或集成测试覆盖关键分支。
