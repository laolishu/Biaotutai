设计：实现方式概述

1. 前端（上传页面）
   - 在上传文件页（或上传模态）新增/更新复选框：`导入时忽略隐藏图层`，UI 文案和现有样式保持一致。
   - 复选框默认状态设为 `checked`（true）。用户可手动取消。
   - 当用户提交上传表单时，将该选项作为 `skipHiddenLayers`（布尔）随请求/事件一并发送到导入处理逻辑。

2. 导入/解析流程
   - 在调用 PSD 解析或 normalize 流程时，使用传入的 `skipHiddenLayers` 参数（如果缺省，则以 true 为默认）。
   - 实现应复用现有 `skipHiddenLayers` 处理逻辑（若已有），确保行为一致：当为 true 时，调用 normalize/parse 时过滤掉 hidden 图层（hidden === true）。

3. 持久化与偏好（可选）
   - 可将用户的上传偏好（是否默认忽略隐藏图层）保存到本地存储或用户设置，以便下次保持一致（非必须，作为可选增强项）。

4. 测试
   - 单元测试上传表单序列化包含 `skipHiddenLayers`。
   - 集成测试：上传带隐藏图层的 PSD，验证勾选时隐藏图层不在结果列表中，取消勾选时隐藏图层存在。
