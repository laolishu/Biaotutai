## 新增需求

### 需求:画布选中图层后面板自动滚动定位
当用户在画布热区点击选中某个图层时，左侧图层面板必须自动将对应的 `LayerItem` 条目滚入可视区域。

#### 场景:画布选中时图层条目滚入可视区
- **当** 用户在画布热区点击某个图层（即 `store.selectedLayerId` 变为该图层 id）
- **那么** 左侧图层面板中对应的 `LayerItem` 元素必须调用 `scrollIntoView({ block: 'nearest', behavior: 'smooth' })` 滚动进入可视区域

#### 场景:已在可视区时不产生不必要滚动
- **当** 对应 `LayerItem` 已在图层面板可视区内
- **那么** 面板禁止发生任何滚动抖动（`block: 'nearest'` 保证此行为）

#### 场景:搜索模式下不触发树视图滚动
- **当** `store.searchQuery` 非空（图层树被隐藏），用户在搜索结果中点击图层
- **那么** 隐藏的 `LayerItem` 不产生可见滚动效果（隐藏元素的 `scrollIntoView` 无副作用）
