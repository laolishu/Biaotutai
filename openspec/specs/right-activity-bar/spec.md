# 右侧 Activity Bar 规格说明

## 需求:右侧图标列必须固定在最右侧
工作台右侧必须提供一个固定宽度的图标列，且该图标列必须位于最右侧边缘，作为右侧分页入口。

### 场景: 页面初始渲染
- **当** 用户首次进入工作台页面
- **那么** 右侧仅显示图标列，图标列位于页面最右侧

## 需求:右侧内容面板默认隐藏并可通过图标切换
右侧内容面板必须默认隐藏；当用户点击图标列中的图标时展开并显示对应分页，再次点击当前激活图标时必须收起。

### 场景: 点击未激活图标展开
- **当** 用户点击右侧图标列中未激活图标
- **那么** 右侧内容面板展开并显示该图标对应分页

### 场景: 点击已激活图标收起
- **当** 用户点击右侧图标列中当前激活图标
- **那么** 右侧内容面板收起并仅保留图标列

### 场景: 展开时恢复最近宽度
- **当** 右侧内容面板处于收起状态后再次被展开
- **那么** 面板必须以最近一次记录宽度展开

## 需求:右侧内容面板支持左边框拖拽调宽
右侧内容面板必须支持通过左边框拖拽调整宽度，宽度范围必须限制在 180 到 420。

### 场景: 拖拽至超出最大值
- **当** 用户向左拖拽导致宽度超过 420
- **那么** 面板宽度必须被限制为 420

### 场景: 拖拽至超出最小值
- **当** 用户向右拖拽导致宽度小于 180
- **那么** 面板宽度必须被限制为 180

### 场景: 拖拽释放后清理监听
- **当** 用户在拖拽结束时松开鼠标
- **那么** 系统必须解除 document 级 mousemove 与 mouseup 监听
